Download the necessary target files before running the souce codes.

For evaluating SOFI performances with different pixel size

1. Run PixesSize1.m
2. Run example.m in file folder 'bSOFI-PixelSize'
3. Run PixelSize2.m
4. Run PixelSize3.m



For evaluating SOFI performances with different noise levels

1. Run Noises1.m
2. Run example.m in file folder 'bSOFI-Noises'
3. Run Noises2.m
